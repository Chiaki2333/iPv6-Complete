# iPv6-Complete
iPv6补全器，将缩写的iPv6完全展开并补全缺失的项。


## Usage
```
Usage: python iPv6-Complete.py [ipv6/Options]

Options:
  -h, --help            Show basic help message and exit

Example:
  python iPv6-Complete.py fe80::e493:3995:5571:be
  python iPv6-Complete.py
```
